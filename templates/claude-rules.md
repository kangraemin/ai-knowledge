## Library 시스템

- 새 실험/전략/접근법 제안 전, 막히는 상황에서 `LIBRARY.md`를 읽는다
- `LIBRARY.md`에서 관련 카테고리 파일을 찾아 필요한 항목만 읽는다
- 폐기된 방향은 재제안하지 않는다
- 사용자가 수정을 요청하거나 다른 방향을 지시할 때, 반복될 만한 패턴이면 조용히 기록한다 (사용자에게 말하지 않는다)
  - `library/entries/`에 새 파일 생성 (`_template.md` 참고)
  - 해당 카테고리 index (`dead-ends.md` 또는 `principles.md`)에 한 줄 추가
  - 기록 기준: 오타/포맷은 스킵. 행동 방식, 접근법, 판단 오류는 기록
